## JavaScript Learning Topics

### Phase 1: Basics (Current)
- Variables (`let`, `const`, `var`)
- Data Types (Primitive & Non-Primitive)
- Operators
- Conditionals (if-else, switch)
- Loops (for, while, for...of, for...in)

### Phase 2: Intermediate
- Functions (Declaration, Expression, Arrow)
- Arrays and Array Methods
- Objects and Object Methods
- DOM Manipulation
- Events
- ES6+ Features (`let/const`, arrow functions, template literals, destructuring, spread/rest)

### Phase 3: Advanced
- Hoisting
- Scope & Closures
- `this` keyword
- Prototypes & Inheritance
- Asynchronous JavaScript (Callbacks, Promises, async/await)
- Fetch API & AJAX
- Modules (import/export)
- Error Handling

### Phase 4: Modern JavaScript
- Classes
- Map, Set, WeakMap, WeakSet
- Optional Chaining & Nullish Coalescing
- Generators & Iterators
- Web APIs (LocalStorage, SessionStorage, Geolocation, etc.)
